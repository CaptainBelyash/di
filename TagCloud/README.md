# Tag Cloud

##
Консольное приложение, создающее облако тегов

###
Usage:

-f FILE 

-out \[FILENAME]

-outpath \[FILEPATH]

-font \[FONT]

-fontsizes \[FONTSIZES]

-imsize \[IMAGESIZE]

-background \[COLOR]

-center \[CENTER]

-step \[STEP]

-pitch \[PITCH]
